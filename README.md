# Quantitative Measures Studio

Interactive static reference site for understanding the quantitative measurement decisions in Micah Miner's mixed methods design.

## Purpose

This site explains and simulates the quantitative strand of a convergent mixed methods study of pedagogical friction and generative AI in K-12 education. It is designed as a learning aid, not as a statistical analysis engine.

The simulations use generated demonstration data only. They do not collect, store, or transmit participant responses.

## What It Covers

- A one-minute explanation of what the quantitative strand contributes to a QUAL + quan design
- Plain-language scripts for explaining the analysis to non-specialists
- Likert-scale item recoding and composite mean scores
- Cronbach's alpha and McDonald's omega as internal consistency estimates
- Why omega is preferred when the item structure supports it
- The N >= 80 gate for exploratory factor analysis
- Small-cell cautions for subgroup comparisons
- Effect-size interpretation alongside statistical significance
- The role of NCES and RAND datasets as national context rather than direct framework measures
- A short glossary and decision-rule table for remembering what each statistical move does
- A linked `comparison/` page adapted from an Antigravity simulation for a more analogy-driven teaching view

## Use

Open `index.html` in a browser, or publish the repository with GitHub Pages from the `main` branch at the repository root.

The comparison page is available at `comparison/index.html`.

## Privacy Note

This is a static educational reference. It does not include analytics, forms, cookies, external scripts, or data storage.
